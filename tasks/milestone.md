## 里程碑 1: 视觉基座与契约 ✅ 已确认

> **目标**: 打牢地基，统一工程规范与物理比例尺
> **优先级**: 最高（必须完全串行）

### 技术选型（已确认）

| 维度 | 决策 |
|------|------|
| **包管理器** | pnpm |
| **前端框架** | Vite + React |
| **3D渲染库** | 原生Three.js |
| **后端框架** | Fastify |

### 任务清单

- [ ] **1.1 Monorepo环境初始化**
  - pnpm workspaces配置
  - 目录结构: `frontend/` `backend/` `shared/` `python-service/`
  - concurrently一键启动脚本
  - Windows环境适配
  - **启动动画**: 赛博朋克终端代码滚动效果

- [ ] **1.2 起草《项目宪法》**
  - `shared/types.ts` 核心类型定义
  - `shared/constants.ts` 坐标常量

- [ ] **1.3 三维摄影机与场景规划**
  - 工作模式: FOV 50, 景深模糊窗外
  - 沉浸模式: FOV 75, 无限焦距
  - **切换方式**: 滚轮切换
  - **窗框设计**: 上下两边有窗框，左右看不到窗框，营造沉浸感
  - **工作台**: 写实桌面（有厚度、边缘、材质）

- [ ] **1.4 低模与后处理管线**
  - Bloom泛光（阈值0.8）
  - Chromatic Aberration（边缘色散）
  - Film Grain（胶片噪点）
  - 玻璃材质折射贴图
  - **后处理强度**: 强烈效果 + 分层后处理技术

- [ ] **1.5 三区城市粗模布景**
  - **建模方式**: HDRI远景 + 低模近景混合
  - **三区比例**: 均等分布（各120度）
  - 过发达区（工业区）: 异形建筑、警示黄灯效、烟尘粒子
  - 发达区（核心区）: 玻璃幕墙、全息垂直绿化
  - 欠发达区（贫民窟）: 错落建筑、霓虹灯牌、洋红/青蓝

- [ ] **1.6 气象系统基础**
  - 3D Rain Particles
  - 玻璃雨痕Normal Map
  - Screen Space Refraction
  - **天气模式**: 随机天气，后期可用AI控制

- [ ] **1.7 日夜循环系统**
  - **时间映射**: 压缩映射（游戏内1小时=现实6小时）
  - 太阳位置计算
  - 天空盒变化
  - 建筑灯光联动

- [ ] **1.8 环境音效系统**
  - 雨声（不同强度）
  - 风声
  - 城市噪音
  - 音量控制

### 验收标准

- [ ] `npm run dev` 一键启动前后端
- [ ] 终端启动动画正常显示（ASCII Logo + 代码流）
- [ ] 滚轮可切换工作/沉浸模式
- [ ] 上下窗框可见（哑光黑色金属），左右无边框
- [ ] 三区城市可见且风格区分明显（赛博朋克2077风格）
- [ ] 随机天气系统正常
- [ ] 后处理效果（Bloom、色差、噪点）强烈且分层
- [ ] 日夜循环正常（压缩映射）
- [ ] 环境音效正常

## 简化决策记录

| 原设计 | 优化后 | 原因 |
|--------|--------|------|
| 多模型路由 | 单一模型 | 简化开发，降低复杂度 |
| 深度音乐联动 | 可选联动 | 用户可控制，降低性能压力 |
| 完整二人世界 | 灯光模式 | 简化实现，保留核心体验 |
| 全部装饰物件 | 3个核心物件 | 聚焦重点，减少工作量 |
