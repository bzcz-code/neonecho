### 2.1 Python图像特征裂变脚本

**目标**: 将2D图片转化为3D渲染所需的深度数据

**具体内容**:

**Base Color Map（底色贴图）**:
- 保持原汁原味的高清色彩
- 作为绒毛根部颜色

**Displacement Map（高度位移图）**:
- 利用Sobel算子提取图片边缘轮廓
- 生成灰度图，让毛毯表面产生高低起伏
- **后续升级**: 可替换为深度学习模型（MiDaS）提高精度

**Roughness Map（粗糙度图）**:
- 智能识别图片中的材质
- 深色区域吸光，浅色区域亮如光纤

**FastAPI接口**:
```python
@app.post("/api/process-image")
async def process_image(file: UploadFile):
    # 1. 读取图片
    # 2. 生成三张贴图
    # 3. 返回贴图URL
    return {
        "base_color": url1,
        "displacement": url2,
        "roughness": url3
    }
```

**验收标准**:
- [ ] 上传图片后成功生成三张贴图
- [ ] Base Color保持原图色彩
- [ ] Displacement正确提取边缘
- [ ] Roughness正确识别材质

### 2.3 赛博珊瑚绒Shader

**目标**: 创建有生命感的毛绒伴侣

**核心设计理念**:
> 用户上传的照片直接成为生命体的遗传图谱。每一根绒毛都从图像像素中汲取生命，不仅保留了原图的所有色彩信息，还赋予了它实体的物理存在和赛博式的光影呼吸感。

**具体内容**:

#### 基础形态映射
- 利用 `InstancedMesh` 生成5万+锥体绒毛
- 每一根从根部像素汲取颜色（Base Color Map）
- 高度由位移图决定
- 线头尖端加入微弱的自发光（Emission）
- **默认位置**: 桌面中央偏前
- **默认尺寸**: 中号（约15cm直径）

#### 波浪式呼吸动画
- Vertex Shader中加入Sine波函数
- 不同区域不同步，产生自然的波浪效果
- 静态毛毯产生轻微的上下浮动与飘动
- **情绪联动**: 开心时呼吸加快，悲伤时呼吸变慢

#### 情绪反馈（颜色+形态）
```glsl
// 情绪参数: 0.0 - 1.0
uniform float emotionScore;
uniform int emotionType; // 0=calm, 1=happy, 2=sad, 3=angry

// 颜色变化
vec3 applyEmotionColor(vec3 baseColor) {
    if (emotionType == 3) { // angry
        return baseColor + vec3(0.2, -0.1, -0.1);
    } else if (emotionType == 1) { // happy
        return baseColor + vec3(0.1, 0.1, -0.05);
    } else if (emotionType == 2) { // sad
        return baseColor + vec3(-0.1, -0.1, 0.2);
    }
    return baseColor;
}

// 形态变化
float applyEmotionDisplacement(float baseHeight) {
    if (emotionType == 1) { // happy - 蓬松
        return baseHeight * 1.2;
    } else if (emotionType == 3) { // angry - 紧绷
        return baseHeight * 0.8;
    }
    return baseHeight;
}
```

#### 交互方式

**鼠标抚摸**:
- Raycaster检测鼠标与绒毛的交互
- 局部顶点偏移模拟"压下"效果
- 缓动函数实现缓慢恢复
- 留下1-2秒的"抚摸印记"
- **AI回应**: 触发简短语音/文字回应（如"好舒服~"）

**语音波纹**:
- AI说话时触发
- 以绒毛球中心为圆心产生水滴样波纹位移场
- 根据音频频段振幅驱动波纹强度

**点击涟漪**:
- 点击时产生涟漪扩散效果
- 波纹从点击位置向外扩散

**拖拽移动**:
- 支持在桌面平面内拖拽移动
- 移动时绒毛自然摆动
- **范围限制**: 仅桌面范围

**验收标准**:
- [ ] 5万+绒毛正常渲染
- [ ] 波浪式呼吸动画自然流畅
- [ ] 情绪变化时颜色+形态都改变
- [ ] 呼吸速度随情绪变化
- [ ] 鼠标抚摸有压下效果

## 技术难点

### 1. InstancedMesh性能优化
5万+实例需要精心优化：
- 使用LOD根据距离减少实例数
- 视锥剔除不可见的绒毛
- 合理的实例数据布局

### 2. Shader复杂度
波浪式呼吸 + 情绪联动 + 多种交互：
- 需要合理的Uniform数量
- 避免过于复杂的计算
- 考虑使用Compute Shader（WebGPU）

### 3. 图像处理质量
Sobel算子可能不够精确：
- 初期使用Sobel算子
- 后期可升级为深度学习模型（MiDaS）
- 或提供用户手动调整工具

### 4. 抚摸AI回应
需要与里程碑3的AI系统集成：
- 抚摸触发简短回应
- 不打断主对话流程
