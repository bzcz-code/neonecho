### 1.3 三维摄影机与场景规划

**目标**: 定义"工作模式"与"沉浸模式"的视线范围

**具体内容**:
- **工作模式**: FOV 50，焦距 0.8m，加入景深模糊窗外
- **沉浸模式**: FOV 75，无限焦距
- **切换方式**: 滚轮切换
- **切换动画**: 贝塞尔曲线平滑过渡
- **窗框设计**: 上下两边有窗框（哑光黑色金属），左右看不到窗框，营造沉浸感
- **飞船舱壁**: 三面环绕（左、右、前），窗户宽大
- **工作台**: 写实桌面（有厚度、边缘、材质）

**技术实现**:
```typescript
// 相机位置配置
const CAMERA_CONFIG = {
  work: {
    position: [0, 1.2, 2],
    fov: 50,
    target: [0, 0.5, 0]
  },
  immersive: {
    position: [0, 1.5, 0.5],
    fov: 75,
    target: [0, 1, -5]
  }
};
```

**验收标准**:
- [ ] 滚轮可切换工作/沉浸模式
- [ ] 切换动画平滑流畅

### 1.7 日夜循环系统

**目标**: 实现完整的日夜变化模拟

**具体内容**:
- **时间映射**: 压缩映射（游戏内1小时 = 现实6小时）
- **太阳位置**: 根据游戏时间计算太阳角度
- **天空盒变化**: 日出/正午/日落/夜晚天空盒切换
- **建筑灯光**: 夜晚自动亮起霓虹灯

**技术实现**:
```typescript
// 时间映射配置
const TIME_CONFIG = {
  // 游戏内1小时 = 现实6小时
  // 游戏内完整24小时 = 现实4天
  timeScale: 6, // 1游戏小时 = 6现实小时
  sunPosition: (gameHour: number) => {
    const angle = (gameHour / 24) * Math.PI * 2 - Math.PI / 2;
    return {
      x: Math.cos(angle) * 100,
      y: Math.sin(angle) * 100,
      z: 0
    };
  }
};
```

**验收标准**:
- [ ] 游戏时间正确映射现实时间
- [ ] 太阳位置随时间变化
- [ ] 天空盒平滑切换
- [ ] 夜晚建筑灯光自动亮起
