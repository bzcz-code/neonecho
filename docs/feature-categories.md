## 一、🤖 AI伴侣核心功能

| 功能 | 决策 | 说明 |
|------|------|------|
| **AI模型** | Gemini 1.5 Pro | 多模态能力强，速度快 |
| **AI人格** | 四种人格（温柔/活泼/幽默/神秘） | 用户可选 |
| **AI声音** | 5-6种可选，按性别/年龄分类 | 个性化选择 |
| **AI形象** | 珊瑚绒即AI形象 | 无独立形象 |
| **AI主动发起** | 可主动（问候/提醒/祝福） | 增强陪伴感 |
| **AI记忆** | 个性化记忆（重要日子、喜好、习惯） | 长期记忆 |
| **AI学习** | 主动学习用户偏好 | 智能适应 |
| **AI回复显示** | 流式输出 | 自然节奏 |
| **AI回复速度** | 自然节奏 | 不机械 |
| **AI回复风格** | 动态调整 | 根据情境 |
| **AI回复长度** | 动态调整 | 根据问题复杂度 |
| **AI上下文** | 完整历史 | 保持对话连贯 |
| **AI情绪触发** | 自动判断 | 智能感知 |
| **AI说话动画** | 需要说话动画（珊瑚绒动画） | 视觉反馈 |
| **AI声音与人格** | 独立选择 | 灵活组合 |
| **语音唤醒** | 仅按钮触发 | 不支持语音唤醒 |
| **对话语言** | 默认自动检测，用户可设置 | 中英文支持 |

## 二、🧸 珊瑚绒系统

| 功能 | 决策 | 说明 |
|------|------|------|
| **珊瑚绒技术** | 照片转毛绒Shader方案 | 核心魔法 |
| **珊瑚绒预设** | 圆形/球形 + 动物造型（猫/狗/熊） | 默认造型 |
| **珊瑚绒位置** | 桌面中央偏前，可拖动 | 灵活布局 |
| **珊瑚绒大小** | 中号（约15cm直径），可调整 | 用户控制 |
| **珊瑚绒命名** | 用户可自定义命名 | 情感连接 |
| **珊瑚绒颜色** | 照片颜色+情绪变化 | 双重影响 |
| **珊瑚绒交互** | 抚摸触发AI简短回应，呼吸速度情绪联动 | 多维反馈 |
| **珊瑚绒特效** | 情绪联动 | 颜色+形态变化 |
| **珊瑚绒进化** | 无进化系统 | 简化设计 |
| **珊瑚绒材质** | 仅毛绒 | 统一风格 |
| **珊瑚绒休眠** | 自动休眠（用户长时间不互动） | 节能设计 |
| **默认珊瑚绒** | 显示默认预设造型 | 引导上传照片 |
| **照片处理失败** | 提示重试 | 友好错误处理 |

## 三、🐱 小猫系统

| 功能 | 决策 | 说明 |
|------|------|------|
| **小猫外观** | 8-bit故障风 + Billboard技术，10+种外观 | 复古赛博风 |
| **小猫获得** | 默认就有 | 无需解锁 |
| **小猫命名** | 用户可自定义命名 | 情感连接 |
| **小猫关系** | 独立存在，不与AI关联 | 独立角色 |
| **小猫行为** | 随机行为 | 自然生动 |
| **小猫睡觉** | 用户不互动时去猫窝（可拖动）睡觉 | 生活化 |
| **小猫互动频率** | 随机行为 | 不可预测 |
| **小猫外观解锁** | 默认全部可用 | 无需解锁 |
| **小猫出现位置** | 从猫窝出现 | 逻辑一致 |
| **小猫音乐联动** | 摇头/跳舞/摇尾巴/眼睛发光 | 音乐响应 |
